Project Description
Cryptocurrency Market Dashboard
A real-time cryptocurrency tracking application built with React that fetches and displays live market data from the CoinGecko API. The application provides users with up-to-date information on various cryptocurrencies including their current prices, market capitalization, and symbols.
Key Features:

Live Data Fetching: Retrieves real-time cryptocurrency market data from CoinGecko API
Dynamic Search Functionality: Filter cryptocurrencies by name or symbol in real-time
Responsive Card Layout: Bootstrap-powered grid system displaying crypto information in organized cards
Visual Representation: Displays cryptocurrency logos/images alongside market data
Error Handling: Graceful error messages and loading states for better user experience
Manual Refresh: Option to manually fetch the latest data with a click
Search Results Counter: Shows the number of filtered results vs total cryptocurrencies.

Technologies Used:

React.js - Frontend framework with hooks (useState, useEffect)
Bootstrap 5 - Responsive UI design and layout
CoinGecko API - Cryptocurrency market data source
Fetch API - Asynchronous data retrieval
CSS - Custom styling

Use Case: 
Perfect for cryptocurrency enthusiasts, traders, or developers who want to quickly monitor multiple cryptocurrencies' market performance in a clean, user-friendly interface without the clutter of complex trading platforms.
